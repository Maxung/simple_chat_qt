# SimpleChatQt

Local chat app using the OpenRouter API.

Tech stack:
- Qt 6.9 with QML
- C++
- md4qt for Markdown parsing and HTML conversion

## Usage/Shortcuts

Set your API key with `export OPENAI_API_KEY=sk-yourkey`.

- CMD+N/CTRL+N = New chat
- CMD+O/CTRL+O = Select model
- CMD+Q/CTRL+Q = Quit
